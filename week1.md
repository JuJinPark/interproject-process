### 기획서

#### 1. 기능

- 자리배치정보
  - 자체 DB 회원정보 활용 자리배치
  - 향우 다른 api 연동을 유념해 설계

- 로그인 기능
  - 하이웍스 api 사용
  - 서버에서 하이웍스 api를 통해 인증후 자체토근을 이용해 프론트와 통신(쿠키와 토큰)
  - 다른 로그인 api로 변경을 유념해 설계 (로그인시 전체 표기에 필요한 정보를 받아서 넣고 상세정보는 매번 api로 통신)

- 검색 기능
  - 이름으로 검색하기
  - 검색결과가 여러개일 시 리스트를 보여주고 선택하기

- 회의실 예약
  - 자체 예약시스템으로 구현
  - 향우 다른 api 연동을 유념해 설계

- 자리 조정 기능
  - 유저별 권한기능 부여후 관리자가 승인하여 자리변경
  - 동일한 자리로 중복요청시 처리
